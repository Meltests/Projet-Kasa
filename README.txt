## KASA APP - Read Me Global ##

Kasa is a fullstack web application built with a **React + Vite** frontend and a **Node.js Express** backend API.


###Front End 

Commands to run the project : npm run dev


###Back End

# Prerequisites
- Make sure that Docker is installed on you device. if not download it from it's website (https://www.docker.com/)

cd /BackEnd
Commands to run the project : docker-compose up -d
Comands to stop the project : docker-compose down
